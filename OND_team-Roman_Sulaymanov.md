## Страница обучения

- При нажатии на первое поле из списка, открывается выбранное поле.

- При нажатии вне модального окна, страница обучения закрывается.
- При нажатии на Esc на клавиатуре, страница обучения закрывается.

#### Страница обучения. Лид-формы. Создать лид-форму.

- При нажатии на "Создать лид-форму" происходит переход на гайд по настройке кампании.

- Создать лид-форму. Загрузка изображения. Изображение, не превышающее размер 256 на 256 пикселей, загружается успешно.

- Создать лид-форму. Загрузка изображения. Изображение, превышающее размер 256 на 256 пикселей, не загружается.

- Создать лид-форму. Название компании. При вводе названия, состощего из любых символов, оно сохраняется и отображается на превью.

<img src='./assets/guide_14.png'>

- Создать лид-форму. Название компании. При вводе слишком короткого названия (0 символов либо пробел), недоступно дальнейшее заполнение.

<img src='./assets/guide_15.png'>

- **Баг** Создать лид-форму. Название компании. При вводе слишком длинного названия (более 30 символов), дальнейшее заполнение доступно.

<img src='./assets/guide_16.png'>

- Создать лид-форму. В лид-форме отображаются введенные названия компаний, заголовков, отображается загруженная фотография.

## Создание аудитории

[Cсылка на раздел](https://ads.vk.com/hq/audience)

### Страница обучения. Создать аудиторию.

- При нажатии на кнопку "Создать аудиторию" появляется модальное окно создания аудитории.

#### Страница обучения. Создать аудиторию. Создание аудитории.

- При нажатии на крестик модальное окно создания аудитории закрывается.

- При нажатии вне модального окна, окно создания аудитории закрывается.

- При нажатии на "Добавить источник" появляется окно "Включить источник" c выбором различных источников.

<img src='./assets/audithory_3.png'>

- При выборе источника происходит его сохранение в аудитории.

<img src='./assets/audithory_2.png'>

- При удалении источника, он удаляется из аудитории.

- При нажатии на "Сохранить" аудитория сохраняется.

- При нажатии на корзину аудитория удаляется.

<img src='./assets/audithory_1.png'>


#### Страница обучения. Создать аудиторию. Создание аудитории. Включить источник.

<img src='./assets/audithory_4.png'>

- При нажатии на "Ключевые фразы" открывается окно с добавлением ключевых фраз, имеющее поля "Название", "Ключевые фразы", "Минус-фразы" и "Период поиска".

<img src='./assets/audithory_5.png'>

- При нажатии на крестик окно ключевых фраз закрывается, при это происходит возврат на страницу "Создание аудитории".

- При нажатии вне модального окна окно ключевых фраз закрывается, при это происходит возврат на страницу "Создание аудитории".

- При попытке сохранить ключевую фразу без заполненных полей сохранения не происходит.

- При вводе в поля "Название", "Ключевые фразы", "Минус-фразы" любого символа, этот символ отображается в выбранном инпуте.

<img src='./assets/audithory_6.png'>

- При вводе любых символов кроме цифр в поле "Период поиска" ввода не происходит.

- При попытке ввести число, не принадлежащее диапазону от 1 до 30, инпут принимает значение 1, если число меньше 1, и 30, если число больше 30.

<img src='./assets/audithory_7_1.png'>
<img src='./assets/audithory_7_2.png'>

- При нажатии на кнопку "Cохранить" новая фраза появляется в списке источников на странице "Создание аудиторий" .

<img src='./assets/audithory_8.png'>
